# ניתוח-רפרנס: Purrweb "Veronica PW" — Designer portfolio (וידאו, 16ש')

> מקור: קובץ-וידאו שאביב סיפק (`Landing - Designer portfolio by Purrweb ... Dribbble.mp4`, 800x600, 16s). פריימים שמורים ב-`reference-frames/`. זה ה"חוקר-רקע" של הריצה (מקור-עיגון חיצוני). הכרעת-אביב לאחר צפייה: **לאמץ הולוגרפי**.
> סוג: ניתוח-ויזואלי מצוטט-מפריימים (`מצוטט`=נראה בפריים, `מוסק`=פרשנות-עיצובית).

## מה זה
תיק-נחיתה של מעצבת-פרילנס ("Veronica PW", "Creative visual designer", "based in London"). קונספט-Dribbble של Purrweb. `מצוטט` (pf_2.5).

## שפת-העיצוב (העדשה המרכזית)
1. **בסיס כהה-פרימיום עריכותי.** רקע near-black עם זוהר-רדיאלי כחול-עמוק עדין + grain קל; הרבה negative-space; scale בטוח וגדול. `מצוטט` (pf_2.5).
2. **טיפוגרפיה דו-קולית:** כותרת-ענק ב-sans נקי (neo-grotesque, Helvetica-like) + **מילת-הדגשה ב-serif-איטלקי** ("Creative *visual* designer", "i create *unconventional*"). תוויות/מטא ב-mono-caps זעיר עם letter-spacing. `מצוטט` (pf_2.5, pf_8).
3. **חתימה הולוגרפית:** כדור-זכוכית **אירידיסנטי** (בועה רפרקטיבית עם קצוות-קשת) מרחף מעל הכותרת בהירו → `מצוטט` (pf_2.5). הוא **מתפוצץ לגרדיאנט-אורורה נוזלי מלא-מסך** (מג'נטה→כתום→ירוק→טורקיז, רווי, אורגני, מטושטש) בסקשן-העבודות. `מצוטט` (pf_8, sheet).
4. **עבודות = דק-כרטיסים כהים על הגרדיאנט:** כרטיסים מעוגלים כהים (near-black) מסודרים כחפיסה/קרוסלה, לפי-קטגוריה (BRANDING / MOBILE & WEB / ILLUSTRATIONS / UX&UI), כל אחד עם **כפתור-`→` עגול**, mockup-של-פרויקט בפנים על גרדיאנט צבעוני, ותווית-ענק. `מצוטט` (pf_11, pf_8).
5. **שובבות/נשמה:** איורי-line-art משחקיים (יד עם עיפרון, כוס-קוקטייל) בצהוב/לבן על הכרטיסים. `מצוטט` (pf_8). מאזן את הרצינות = "מוצר עם נשמה".
6. **תנועה:** ההירו-הכהה **נמס** לתוך הגרדיאנט-הצבעוני; טקסט-שמאל sticky/parallax; כרטיסים נפתחים/מחליקים; cursor-חץ. סינמטי וחלק. `מוסק` (מרצף-הפריימים בsheet).

## מה זה אומר לאביב (גשר להכרעה)
- **קוהרנטי עם ההירו הקיים:** ההירו של אביב כבר כהה עם מקור-אור-מתפשט (נורת-אמבר) — מבנית זהה ל"מקור-הולוגרפי-שמתפשט" של הרפרנס. → המנגנון נשאר, הצבע הופך אירידיסנטי.
- **פותר את הכרטיסים:** "כרטיסים כהים זוהרים על אטמוספירה" = בדיוק מה שצריך להחליף את הקופסאות-הברוטליסטיות.
- **טיפוגרפיה:** sans+serif-איטלקי = מהלך עריכותי שמוסיף נשמה (אביב כיום Space Grotesk+Mono; להוסיף serif-איטלקי לאקצנט).

## המתח לזכור (מול סינתזת-המשרות)
הרפרנס **אקספרסיבי/אמנותי** (תיק-פרילנס), בעוד המשרות רוצות **בהירות-מוצרית/design-system**. הסכנה: גרדיאנט-הולוגרפי גנרי שנראה כמו עוד תיק-טרנדי. הפתרון: הבסיס שיטתי-מרוסן (טוקנים, היררכיה, ריסון) = הריגור-המוצרי; ההולוגרפי+serif+line-art = הבידול/נשמה.
