# שחרור-עקרונות (אביב, 2026-06-26) — דורס את spec-v2 לטובת 1:1 עם הרפרנס

אביב זיהה ש**עקרונות-יסוד קשיחים חוסמים עיצוב 1:1**. הכרעה: לשחרר 3, לשמור 1.

## נשמר (לא שוחרר)
- **הפורטרט + המנורה בהירו נשארים.** ההירו לא יהיה 1:1-מבני עם הרפרנס (שהוא טיפו-בלבד+כדור) — זה מקובל. ההירו = היברידי: טיפו-ענק + פורטרט-משולב + מנורה.

## שוחרר (override)
1. **🔓 "מסכים-אמיתיים / רק-פרויקטים-גולמיים" בכרטיסים** → מותר **line-art שובב / סטיילינג-עיצובי / כרטיסי-קטגוריה / mockup-על-גרדיאנט**, כמו הרפרנס. (מבטל חלקית את "אין דמה" לטובת מצגת-עיצובית; התוכן עדיין הפרויקטים האמיתיים, רק הטיפול ויזואלי-נועז.)
2. **🔓 P1 "בהירות-מוצרית גוברת / ריסון"** → **מקסימליזם ואקספרסיביות מובילים.** רווי, שכבתי, נועז כמו הרפרנס. (הריגור-המוצרי כבר לא הרצפה הקשיחה.)
3. **🔓 "מנגנון-התנועה הקדוש (GSAP pinned-scrub)"** → **מותר לבנות-מחדש** את הגלילה/זום/מעבר מאפס כדי להתאים בדיוק לרפרנס (במקום לשמר את הקיים).

## השלכות-מימוש
- כרטיסי-עבודות: לעצב-מחדש נועז (label-ענק + → + media-על-גרדיאנט-חי + עומק/הנפה), אולי line-art.
- אורורה: יותר רווי/שכבתי/בהיר.
- תנועה: לבנות-מחדש לפי הצורך אם הקיים לא מספיק-מדויק (מאומת חי ע"י אביב).
- **אזהרה שנותרה:** ככל שמתרחקים ל"אקספרסיבי", פוחת אות-ה"design-system/clarity" שהמשרות ביקשו ([[reference_target-jobs-2026]]). אביב בחר ביודעין 1:1>משרות לעת-עתה.
