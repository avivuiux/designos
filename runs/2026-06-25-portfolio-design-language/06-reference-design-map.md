# מפת-הרפרנס המלאה — Purrweb "Veronica PW" (לבנייה 1:1)

> נדגם מהווידאו (פריימים ב-`reference-frames/`) + דגימת-פיקסלים (PIL). זה ה-ground-truth ל-1:1.
> הכרעת-אביב: **אימוץ מלא, כל האתר, בלי יוצא מן הכלל.** שומרים את הפורטרט + המנורה. קריאוּת = לא-חוסם (אביב אישר).

## 1. צבעים (מדגימת-פיקסלים)
- **בסיס:** כמעט-שחור-מוחלט `#05050b` (דגימות: `#000000`,`#010303`). רמז-כחול-קר עדין למעלה (`#153445`). **לא** אפור-כהה - שחור.
- **אורורה (קשת-נוזלית חיה, פריחות-תכשיט על שחור):**
  - טורקיז/ירוק-זוהר — הכי בולט: שיא `#0bffbf` → גוף `~#10d9a8`.
  - אמבר/זהב: `~#e8a93a` (דגימה `#e4ad30`).
  - מג'נטה/ורוד: שיא `~#ff2f9e` (כהה-קורא `#4a0036`).
  - סגול/violet: `~#7a3fe0`.
  - נגיעת כחול קר למעלה.
  - **עיקרון:** השחור תופס את רוב-השטח; הצבעים פורחים בכתמים רוויים. (זה מה שצריך לתקן אצלי - הבסיס חייב להיות שחור-יותר, הצבעים רוויים-יותר.)
- **טקסט:** לבן `#ffffff` / כותרות; גוף לבן-עמום.

## 2. פונטים
- **כותרות (display):** גרוטסק נקי וניטרלי (סגנון Helvetica/Aeonik) — **לא** Space Grotesk (שהוא יותר אופייני/quirky). משקל בינוני (~500-600), tracking הדוק, line-height ~0.95.
  → התאמה-חופשית קרובה: **Inter Tight** (או Hanken Grotesk / Geist).
- **אקצנט:** serif-איטלקי בקונטרסט-גבוה ("visual", "unconventional"), לרוב בגוון חם/זהב. → **Instrument Serif** italic (כבר נטען) = התאמה טובה.
- **תוויות/מטא:** caps עם letter-spacing ~0.15-0.2em, ~11px, אטימות-נמוכה. (גרוטסק-caps או mono.)

## 3. טיפוגרפיה (טיפול)
- כותרת-ענק `clamp(3.5rem, 7vw, 7rem)`, מילת-אקצנט ב-serif-איטלקי, מעט גדולה-יותר ובגוון-חם.
- מטא: mono/grotesk caps, tracking רחב, אטום ~45%.
- היררכיה דרך גודל+משקל+אטימות, לא צבע-קופץ.

## 4. אנימציות
1. **הירו:** הכותרת מתיישבת ב-load; **כדור-זכוכית הולוגרפי** מרחף/מסתובב עדין ושובר-אור מעל "designer".
2. **גלילה הירו→עבודות:** ההירו-הכהה **נמס** - האורורה פורחת פנימה (הצבעים מתרחבים), הכותרת מתגדלת+נמוגה.
3. **עבודות:** טקסט-פתיחה שמאלי **דביק (sticky)**; חבילת-כרטיסים ימנית **מחליקה אופקית** בגלילה (כרטיסים נכנסים מימין, מדורג). נקודות-pagination למטה.
4. **כרטיס:** ב-hover/כניסה — מתרומם/zoom-עדין על ה-mockup.
5. לולאה חוזרת להירו. cursor-חץ מותאם.

## 5. קומפוזיציה (per section)
- **הירו:** nav (לוגו שמאל / לינקים ימין) · כותרת-ענק center-left · מטא-mono בפינות (זמינות/מיקום) · כדור-הולוגרפי מרחף ימין-מרכז.
- **עבודות:** 2-עמודות — פתיחה-דביקה שמאל (~⅓) + חבילת-כרטיסים ימין (~⅔) · נקודות למטה.
- כל-האתר: בסיס-שחור + אורורה-חיה כרקע משותף, הרבה negative-space, scale-בטוח.

## 6. מה אצל אביב שונה-במכוון (נשמר)
- **פורטרט + מנורה** = מרכז-ההירו (במקום טיפוגרפיה-בלבד + כדור). המנורה = מקור-האור שפורץ-לאורורה (תפקיד הכדור-ההולוגרפי).
- תוכן אמיתי (Bezeq/Mykey/PriceMatch) במקום קטגוריות-דמה.
- מנגנון-ההירו (GSAP) נשמר; רק הצבע/הסטייל מתלבשים עליו.

## 7. פערים מול המימוש הנוכחי שלי (לתיקון ל-1:1)
- בסיס `--color-night` בהיר מדי (`#0b0b12`) → לשחור (`#05050b`).
- אורורה לא-מספיק-רוויה/חיה + חסר טורקיז-זוהר דומיננטי.
- פונט-display = Space Grotesk → לשקול Inter Tight (החלטת-אביב).
- חסר: כדור/אלמנט-הולוגרפי, sticky-אמיתי, החלקת-כרטיסים-אופקית-בגלילה, נקודות-pagination.
